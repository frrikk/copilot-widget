# Next.js Standard Example (Mode A)

Uses `<ChatWidget>` with the default WebChat UI and `styleOptions` for theming.

## Setup

```bash
npm install
```

Create `.env.local`:

```
COPILOT_SECRET=your-direct-line-secret
```

## Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

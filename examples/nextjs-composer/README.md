# Next.js Composer Example (Mode B)

Uses `<CopilotComposer>` with a fully custom chat UI built using your own components.
All WebChat hooks (`useActivities`, `useSendMessage`, etc.) are available inside the Composer.

## Setup

```bash
npm install
```

Create `.env.local`:

```
COPILOT_SECRET=your-direct-line-secret
```

## Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
